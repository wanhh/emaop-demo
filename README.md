## emao-plus demo

emao-plus的demo。其中会包含常用异构语言的使用、目录规范的制定、压缩合并等等；

###目录规范
    - src为开发目录，
    - 开发人员只需要维护src目录即可
    - 开发完成之后执行build.bat会自动生成build目录
